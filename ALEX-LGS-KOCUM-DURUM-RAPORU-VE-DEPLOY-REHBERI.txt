3. NETLIFY'A DEPLOY ETME REHBERİ (BAŞLANGIC SEVİYESİ)
========================================================================

ADIM 1: HESAP OLUŞTURMA
1. https://netlify.com adresine git
2. "Sign up" butonuna tıkla
3. GitHub hesabınla giriş yap (yoksa önce GitHub hesabı oluştur)
4. E-posta adresini doğrula

ADIM 2: PROJE DOSYALARINI HAZIRLA
1. v0.dev'den projeyi indir (ZIP dosyası olarak)
2. ZIP dosyasını bilgisayarında aç
3. Klasör adını "alex-lgs-kocum" olarak değiştir

ADIM 3: NETLIFY'A YÜKLEME
1. Netlify ana sayfasında "Add new site" butonuna tıkla
2. "Deploy manually" seçeneğini seç
3. Proje klasörünü sürükle-bırak alanına at
4. "Deploy site" butonuna tıkla
5. 2-3 dakika bekle, site otomatik olarak yayınlanacak

ADIM 4: DOMAIN AYARLAMA (İSTEĞE BAĞLI)
1. Site yayınlandıktan sonra "Domain settings" bölümüne git
2. "Change site name" ile özel isim ver (örn: alex-lgs-kocum)
3. Sitenin adresi: https://alex-lgs-kocum.netlify.app olacak

ADIM 5: GÜNCELLEME YAPMA
1. Değişiklik yapmak istediğinde
2. Yeni dosyaları tekrar sürükle-bırak
3. Otomatik olarak güncellenecek

========================================================================
4. GITHUB'A DEPLOY ETME REHBERİ (BAŞLANGIÇ SEVİYESİ)
========================================================================

ADIM 1: GITHUB HESABI OLUŞTURMA
1. https://github.com adresine git
2. "Sign up" butonuna tıkla
3. Kullanıcı adı: "ozkan-alex-lgs" gibi bir isim seç
4. E-posta ve şifre belirle
5. Hesabı doğrula

ADIM 2: YENİ REPOSITORY OLUŞTURMA
1. GitHub'da "+" işaretine tıkla
2. "New repository" seç
3. Repository adı: "alex-lgs-kocum"
4. "Public" seçeneğini işaretle
5. "Add a README file" kutusunu işaretle
6. "Create repository" butonuna tıkla

ADIM 3: DOSYALARI YÜKLEME
1. Oluşturulan repository sayfasında "uploading an existing file" linkine tıkla
2. Proje dosyalarını sürükle-bırak alanına at
3. Commit message: "İlk versiyon - Alex LGS Koçum"
4. "Commit changes" butonuna tıkla

ADIM 4: GITHUB PAGES AKTIFLEŞTIRME
1. Repository sayfasında "Settings" sekmesine git
2. Sol menüden "Pages" bölümünü bul
3. Source: "Deploy from a branch" seç
4. Branch: "main" seç
5. "Save" butonuna tıkla
6. 5-10 dakika sonra site hazır olacak

ADIM 5: SİTE ADRESİNİ BULMA
1. Settings > Pages bölümünde
2. Site adresi görünecek: https://ozkan-alex-lgs.github.io/alex-lgs-kocum
3. Bu adresi kaydet ve paylaş
